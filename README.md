# Source_Tract_Coupling
This program is written for c-language. It includes

SourceTractModel.c : Simulate vocal fold oscillations coupled to vocal tract acoustics.

SubglottalAreaF.txt : Area function of subglottis

Running this code will reproduce the results in the publication :
I. T. Tokuda, M. Zemke, M. Kob, and H. Herzel, "Biomechanical modeling of register transitions and the role of vocal tract resonators," Journal of the Acoustical Society of America, Vol. 127, Issue 3, pp. 1528-1536 (2010); C. T. Herbst, C. P. H. Elemans, I. T. Tokuda, V. Chatziioannous, J. G. Svec, "Dynamic system coupling in voice production," Frontiers in Frontiers in Network Physiology (2022)
